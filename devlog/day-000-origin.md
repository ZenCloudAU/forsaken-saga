# Day 000 — The Origin

The Forsaken Saga begins as both a game and a record of learning.

This repository is not only for source code. It is the workshop floor, the world bible, the learning archive, the design journal, the sprint tracker, the production diary, and the public build record.

The first decision was deliberate restraint.

Before building combat systems, AI systems, lore engines, procedural worlds, or advanced mechanics, the project establishes the operating model first.

The initial prototype target is intentionally small:

One room.

One NPC.

One visible state mechanic.

One dialogue choice.

One consequence.

The project is being built publicly while learning Godot and GDScript in real time.
