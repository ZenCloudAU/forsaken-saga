# Sprint 00 — Studio Foundation

## Objective

Establish the production foundation before major game development begins.

## Scope

Create the repository operating model.

Create the GitHub Pages shell.

Create the Obsidian-style vault structure.

Create the project roadmap.

Create the tool register.

Create the command log.

Create the architecture decision system.

Create the first public devlog.

Install and understand Godot.

Complete the first Godot orientation tutorial.

## Not In Scope

No advanced combat systems.

No inventory system.

No procedural generation.

No advanced shaders.

No 3D.

No monetisation.

No Patreon.

No AI NPC systems.

No production art pipeline.

## Exit Criteria

The repository can explain itself.

The project has a visible structure.

The GitHub Pages shell is live.

The first devlog exists.

The first roadmap exists.

The first architecture decision records exist.

Phil understands the basic Godot workflow.
